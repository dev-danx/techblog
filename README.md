# My Blog

A simple blog powered by [GitHub Pages](https://pages.github.com) and [Jekyll](https://jekyllrb.com).

## Getting Started

1. **Fork this repo** and rename it to `yourusername.github.io`
2. **Edit `_config.yml`** with your details
3. **Add posts** to `_posts/` folder (format: `YYYY-MM-DD-title.md`)
4. **Push to main** — your blog will be live!

## Writing Posts

```markdown
---
layout: post
title: "My Post Title"
date: 2026-04-08
categories: misc
---

Your content here...
```

## Local Development

```bash
gem install jekyll bundler
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`
